Getting Started

First-time setup

No sign-up, no permissions to configure. Install and start saving networks immediately.

Install the APK

Download and install the signed APK on your Android device. Allow installation from unknown sources in Settings → Security if prompted.

Add your first network

Tap + Add Network and choose your method: enter details manually, scan a QR code, or paste a QR string.

Save and view

Your saved network appears in the list immediately. Tap any entry to view the full credentials including the password.

Edit or delete anytime

Long-press or tap the edit icon on any entry to update its details. Tap delete to remove it permanently.

QR Codes

Working with QR codes

Two ways to add a network from a QR code — with a camera or without.

📷

Scan a QR Code

Tap Scan QR Code and point your camera at any Wi-Fi QR code — typically found on router stickers, café menus, or home network labels. The app parses the SSID, security type, and password automatically and prompts you to save.

📋

Import a QR String

If you have a Wi-Fi QR string as text (e.g. copied from another app or document), use Import QR String and paste it directly. The app decodes the string and fills in the credentials for you to review and save.

QR string format: WiFi Connect supports the standard Wi-Fi QR code format: WIFI:S:<SSID>;T:<WPA|WEP|nopass>;P:<password>;H:<true|false>;
Free plan scan limit: Camera scanning is limited to 7 scans per week and 2 scans per day. After the daily or weekly limit is reached, a 24-hour cooldown applies before you can scan again. Gallery QR import has no limit and can be used freely at any time. Upgrade to WiFi Connect Pro for unlimited camera scans — see Plans.

Backup & Restore

Backup & Restore

WiFi Connect keeps everything local. Backup & Restore gives you full control over saving and recovering your network list — without any cloud dependency.

📦

Save Private App Backup

Tap Save Private App Backup in the Backup & Restore menu to write a JSON backup into the app's private storage. Each save also writes a stable restore file named wifi-connect-private-backup.json, so the Restore function always knows exactly where to look — no browsing required.

🔄

Restore Private Backup

Tap Restore Private Backup to load the backup that was saved to app storage. This is the fastest way to recover your list after a reinstall or data loss — as long as the app storage file is still present.

Private backup location: App storage is sandboxed to WiFi Connect. The file is not directly browsable by other apps. Use Import JSON from File if you need to restore from a backup you copied elsewhere.

FAQ

Frequently asked questions

Is my data stored online or in the cloud?
No. All saved network names and passwords are stored locally on your Android device only. The App makes no network requests and never transmits any data externally.
Does WiFi Connect actually connect to Wi-Fi for me?
WiFi Connect is a credential manager and organiser — it saves and displays your Wi-Fi credentials. On most Android versions (10+), third-party apps cannot programmatically join networks without user interaction. Use the saved password to connect manually through your Android Wi-Fi settings.
What happens if I delete the app?
Uninstalling removes all saved credentials and the private app backup from your device. Before uninstalling, export a JSON backup to somewhere outside the app — free users can do this via Save Private App Backup then copy the file, or Pro users can use Share / Export to send it to cloud storage or another app. After reinstalling, use Import JSON from File to restore your list.
What QR code formats are supported?
WiFi Connect supports the standard Wi-Fi QR code format: WIFI:S:<SSID>;T:<WPA|WEP|nopass>;P:<password>;H:<true|false>;. This is the format used by most routers, Android share-to-QR, and iOS share-to-QR features.
Can I import a QR string from another app?
Yes. If another app or document gives you a Wi-Fi QR string as text, use the Import QR String feature and paste it. The App will decode it and let you save the credentials.
What does App Lock protect?
App Lock is a Pro feature that adds three layers of protection using your device PIN:

1. App entry — your PIN is required every time the app is opened or resumed from the background.
2. Export protection — no JSON or CSV export can be triggered without entering your PIN first.
3. Copy & share protection — copying a password or sharing credentials requires a successful PIN unlock.

All authentication is handled entirely by your device's operating system. WiFi Connect does not store or transmit your PIN.
What is WiFi Connect Pro?
WiFi Connect Pro is the optional paid upgrade available through Google Play. It unlocks unlimited QR camera scans, external JSON/CSV sharing, and App Lock — PIN protection for the app, exports, and credential copying. The free plan remains fully functional with no ads — Pro is for users who need higher scan volume or want to back up their list externally.
What happens when I reach the weekly QR scan limit?
Free users can perform 7 QR camera scans per week with a maximum of 2 scans per day. After the daily or weekly limit is reached, a 24-hour cooldown applies before you can scan again. During the cooldown you can still add networks manually, import from gallery QR images, or paste QR strings — none of these count toward the limit. Upgrade to WiFi Connect Pro to remove the limit entirely.
Does the QR scan limit apply to gallery import?
No. The 7 scans/week (2/day) limit applies to live camera scanning only. Importing a QR code from your gallery, pasting a QR string, or adding a network manually are all unlimited on the free plan.
Is there a limit to how many networks I can save?
There is no hard limit set by the App. The practical limit is determined by your device's available local storage, which is typically large enough to store thousands of entries.
Can I export or back up my saved networks?
Free users: Use Save Private App Backup to write a JSON backup to app storage, then Restore Private Backup to recover it anytime. You can also use Import JSON from File to restore from a backup you've copied elsewhere.

Pro users: All of the above, plus external JSON/CSV sharing to other apps, cloud storage, or email via the header export buttons and share sheet. CSV is for viewing only and cannot be reimported.
